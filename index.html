installer SASS

Sur Ubuntu

Installer ruby :

sudo apt install ruby

Installer ruby-sass :

sudo apt-install ruby-sass

Sur MacOs

Ruby est nativement installé, il suffit donc d'installer SASS :

gem install sass

Sur Windows

Installer ruby :

Utilisez ruby installer

Installer SASS :

gem install sass

Compiler un fichier .scss en .css

On crée un fichier style.scss, dans lequel on va écrire notre code en scss.
Les navigateurs ne lisant pas les fichiers .scss, il nous faut le compiler en css, avec cette commande :

sass --watch style.scss:style.css

La syntaxe SCSS

L’une des caractéristiques les plus intéressantes de Sass est la possibilité d’imbriquer des règles. Dans un fichier CSS habituel, les règles (sélecteur/propriété/valeur) sont listées à la suite et chaque sélecteur doit inclure tous ses éléments :

//code CSS habituel
body.home .media-unit {
border: 1px solid #ccc;
background-color: #fff
}
body.home .media-unit .right {
border-left: 1px solid #ccc;
}
body.home .media-unit .right h1 {
font-size: 24px;
}

Utiliser les variables en Sass

La déclaration de variables est très simple en Sass, la syntaxe est celle ci :

$maVariableCouleur: #be608b

Vous pouvez déclarer votre variable au début de votre ficher .scss ou dans un fichier .scss distinct, dédié à contenir toutes vos variables (comme nous le verrons au chapitre consacré aux imports.

Vous pouvez ensuite l'appeler n'importe où, et autant de fois que vous le voulez, dans votre SCSS de la manière suivante :

h1{
color: $maVariableCouleur;
}
