Tutoriel SASS

installer SASS

Sur Ubuntu

Installer ruby :
sudo apt install ruby

Installer ruby-sass :
sudo apt-install ruby-sass

Sur MacOs

Ruby est nativement installé, il suffit donc d'installer SASS :
gem install sass

Sur Windows

Installer ruby :
Utilisez ruby installer

Installer SASS :
gem install sass

Compiler un fichier .scss en .css

On crée un fichier style.scss, dans lequel on va écrire notre code en scss.
Les navigateurs ne lisant pas les fichiers .scss, il nous faut le compiler en css, avec cette commande :
sass --watch style.scss:style.css
